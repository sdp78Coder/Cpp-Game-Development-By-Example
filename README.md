# Cpp-Game-Development-By-Example
- First section of the book, I will create a 2D game, initially with Simple and Fast
Multimedia Library (SFML), which covers the basics that are required to create any game,
and with which we can make any game with the utmost ease, without worrying about
how a game object is drawn. I will be using SFML 2.5.1 just to draw our game objects.
